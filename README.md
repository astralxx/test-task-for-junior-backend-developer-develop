# 📋 Task Service

✅ Выполнил тестовое задание.  
Сервис для управления задачами с поддержкой **периодических задач** для персонала.

---

## 🚀 Быстрый запуск (Docker Compose)

### Требования

- Go 1.23+
- Docker и Docker Compose

### Запуск

```bash
docker compose up --build
```

### Если PostgreSQL уже запускался раньше со старой схемой

```bash
docker compose down -v
docker compose up --build
```

---

## ✨ Возможности

- 🧩 CRUD операции над задачами  
- 🔁 **Периодические задачи** (4 типа):  
  - 📅 **Ежедневные** — каждые N дней  
  - 📆 **Ежемесячные** — в указанное число месяца (1–30)  
  - 🔢 **По чётным / нечётным дням** месяца  
  - 🎯 **На конкретные даты** — только в выбранные дни  
- ⚙️ Автоматическое создание всех экземпляров задачи при создании шаблона  
- 📄 Swagger-документация

---

## 🧠 Как работает периодичность

При создании задачи с полем `recurrence_type` сервис **автоматически генерирует** все будущие экземпляры:

- Если `recurrence_end_date` не указан → задачи создаются на **1 год вперёд**.
- Каждый экземпляр — это **полноценная независимая задача** (можно отдельно менять статус, удалять и т.д.).
- Поля `recurrence_*` сохраняются в каждой задаче — видно, по какому правилу она была создана.
